# ImageMatch

一个基于 MFC + OpenCV 的桌面图像配准与拼接工具，面向大分辨率图像场景，支持多种刚性配准方法、配准日志输出、结果拼接，以及特征点和匹配关系可视化。

## 功能概览

- 支持加载 `Source` 和 `Target` 两张图像，支持拖拽导入。
- 支持多种配准方法：
  - `PHASE`：基于相位相关的快速平移配准。
  - `ECC`：基于强度优化的非特征点刚性配准。
  - `ORB`：轻量级特征点配准。
  - `AKAZE`：鲁棒特征点配准。
  - `SIFT`：更强鲁棒性的特征点配准。
- 支持输出刚性变换矩阵、旋转角、平移量、RMSE、关键点数、匹配数、内点数等结果。
- 支持大图加速：针对超大分辨率图像自动降低工作分辨率、限制特征点数量与匹配数量。
- **UI增强功能**：
  - 在每个窗格标题旁显示图像分辨率（宽 x 高格式）
  - 特征点方法显示匹配点连线（青色线段，橙色端点）
  - 配准完成后显示Toast提示（成功绿色、失败红色）
- 支持特征点结果可视化：
  - 在 `Source` 窗格显示源图关键点和目标点反投影连线。
  - 在 `Target` 窗格显示目标图关键点和源点正向投影连线。
- 支持拼接结果预览，并提供缩放、平移查看能力。

## 大图优化策略

项目针对高分辨率图像做了专门优化，避免直接在原始分辨率下执行高成本配准：

- 自动识别大图与超大图场景。
- `PHASE`：
  - 使用更激进的降采样，优先快速估计平移。
- `ECC`：
  - 使用相位相关结果作为初值。
  - 在大图模式下降低工作尺寸与最大迭代次数。
- `ORB / AKAZE / SIFT`：
  - 自动降低工作图尺寸。
  - 自动限制特征点数量。
  - 自动限制保留匹配对数量。

对于超大图，推荐优先尝试：

1. `PHASE`
2. `ECC`
3. `ORB`

如果图像纹理复杂且需要更强鲁棒性，再尝试 `AKAZE` 或 `SIFT`。

## 日志输出

每次配准都会输出详细日志，显示在结果文本框中，同时通过 `TRACE` 输出到调试窗口。日志包含：

- 输入图像尺寸
- 加速策略说明
- 工作分辨率与缩放比例
- 灰度转换耗时
- 特征提取耗时
- 匹配与 RANSAC 耗时
- ECC / 相位相关耗时
- RMSE 计算耗时
- 拼接耗时
- 总耗时

便于分析不同算法在大图下的速度表现。

## 界面说明

- `Source`：源图窗口
- `Target`：目标图窗口
- `Stitched`：拼接结果窗口
- 方法按钮：
  - `PHASE`
  - `ORB`
  - `AKAZE`
  - `SIFT`
  - `ECC`
- `Reset`：重置当前视图缩放与平移

交互方式：

- 鼠标左键拖拽：平移图像
- 鼠标滚轮：围绕光标缩放
- 拖拽文件到 `Source` 或 `Target` 窗口：直接加载图像

## UI增强功能

### 分辨率显示
每个窗格的标题右侧显示当前图像的分辨率：
- Source / Target 窗格：显示已加载图像的分辨率（"W x H"格式）
- Stitched 窗格：显示拼接结果分辨率，或"No image"当未加载

### 特征点匹配可视化
使用 ORB、AKAZE 或 SIFT 配准时：
- 在 Source 窗格中显示匹配连线（青色线段）
- 在 Target 窗格中显示匹配连线（青色线段）
- 匹配端点用橙色圆点标记
- 支持鼠标缩放和平移时的正确坐标变换

### 配准操作提示
点击配准按钮时的完整UI流程：
1. 清空 Stitched 窗格中的上一次结果
2. 显示 "Running [方法名] ..." 运行状态
3. 配准完成后显示 Toast 提示：
   - **成功**：绿色 "Registration Succeeded" （1.8秒后消失）
   - **失败**：红色 "Registration Failed" （1.8秒后消失）

## 使用流程

1. 加载或拖拽源图和目标图。
2. 点击一种配准方法。
3. 查看结果框中的：
   - 配准状态
   - 耗时
   - 工作缩放比例
   - 关键点/匹配/内点数量
   - 刚性变换矩阵
   - 详细日志
4. 若使用特征点方法，可在 `Source` / `Target` 窗格中查看特征点和匹配连线。
5. 在 `Stitched` 窗格中查看拼接结果。

## 当前实现说明

- 当前配准模型以刚性/欧式变换为主，适用于平移加小角度旋转场景。
- `PHASE` 和 `ECC` 属于非特征点方法，适合速度优先场景。
- `ORB / AKAZE / SIFT` 属于特征点方法，适合纹理较丰富、需要更强匹配鲁棒性的场景。

## 技术细节

### 图像拼接算法
- 使用仿射变换将源图像映射到目标坐标系
- 计算联合边界框以确定canvas尺寸
- 采用加权平均（50% + 50%）混合重叠区域
- 在非重叠区域直接复制目标图像像素

### 特征点可视化
- 绿色圆点：特征点匹配对中的源图特征点
- 橙色圆点：目标图像上映射后的对应点
- 青色连线：连接匹配对的线段
- 支持实时缩放和平移，坐标自动变换

### 性能优化
- 自适应降采样：根据图像尺寸自动调整工作分辨率
- 分层处理：使用图像金字塔加速优化过程
- 特征点限制：自动限制保留的匹配对数量以控制内存使用
